| Report Generator | Court-ready PDF reports + Excel evidence export |

```mermaid
graph TD
    A[Next.js Web App] --> E[Recovery API]

    F2 --> F3[3. Layer Resolver - 7-Layer Historical FS Detection]
```

- RAID assembler for multi-disk recovery scenarios
